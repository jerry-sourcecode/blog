## Web 内容渲染

### 随机引用

`Quote` 组件是一个简单的组件，用于显示随机引用。

<MdxRepl>
  <Quote />

  <Fragment slot='desc'>
    ```jsx
    import { Quote } from 'astro-pure/advanced'

    <Quote />
    ```
  </Fragment>
</MdxRepl>

配置：

```ts title="src/site.config.ts"
export const integ: IntegrationUserConfig = {
  // 在页脚添加随机引用（默认在首页页脚）
  quote: {
    // https://github.com/lukePeavey/quotable
    server: 'https://api.quotable.io/quotes/random?maxLength=60',
    target: `(data) => data[0].content || '错误'`
  }
}
```

[Quotable](https://github.com/lukePeavey/quotable) 是一个提供随机英文引用的开源 API。

对于中文引用，你可以使用 [Hitokoto](https://developer.hitokoto.cn/sentence/)。

```ts title="src/site.config.ts"
export const integ: IntegrationUserConfig = {
  quote: {
    // https://developer.hitokoto.cn/sentence/#%E8%AF%B7%E6%B1%82%E5%9C%B0%E5%9D%80
    server: 'https://v1.hitokoto.cn/?c=i',
    target: `(data) => data.hitokoto || '错误'`
  }
}
```

### 链接预览

预览内容中插入的任何链接。

<MdxRepl width='80%'>
  <LinkPreview href='https://www.mozilla.org/en-US/' />
  <LinkPreview href='https://www.cloudflare.com/' hideMedia />

  <Fragment slot='desc'>
    ```jsx
    import { LinkPreview } from 'astro-pure/advanced'

    <LinkPreview href='https://www.cloudflare.com/' />
    // `zoomable` 选项将控制 mediumzoom 标签
    // <LinkPreview href='https://www.cloudflare.com/' zoomable={false} />
    <LinkPreview href='https://www.cloudflare.com/' hideMedia />
    ```
  </Fragment>
</MdxRepl>
