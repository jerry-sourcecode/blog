组件让你能够轻松地一致复用一段 UI 或样式。你不仅可以在 `.astro` 文件中使用它们，还可以在 `.mdx` 文件中使用。

<Aside type='tip'>
[MDX](https://mdxjs.com/) 是一种让你在 Markdown 中嵌入 JSX 的格式。除了这一点，它在其他方面与 markdown 文件没有区别。
</Aside>

对于 `.astro`，你可以直接导入并使用组件。第一部分也会展示一个示例。

<Aside type='tip'>
如果你总是在内容中导入一些组件，你可以使用 [astro-auto-import](https://github.com/delucis/astro-auto-import/tree/main/packages/astro-auto-import) 来自动导入它们，而无需在每个文件中手动导入。
</Aside>


<Fragment slot='desc'>
  ````jsx
  import { MdxRepl } from 'astro-pure/user'

  // width 是可选的；使用 width 参数设置
  // MDX Repl 组件内的所有元素。
  // （desc 插槽不受此参数影响）
  <MdxRepl width='100%'>
    <p>你好</p>
    <Fragment slot='desc'>
      ```html
      <p>你好</p>
      ```
    </Fragment>
  </MdxRepl>
  ````
</Fragment>
</MdxRepl>

你可以为 `desc` 插槽组合任何其他组件。`<Fragment>` 不会作为父 HTML 容器标签被渲染。
