## 贡献代码

向所有为 [astro-theme-pure](https://github.com/cworld1/astro-theme-pure) 做出贡献的人们表示衷心的感谢。开源社区因你们而精彩。

[![contributors](https://contrib.rocks/image?repo=cworld1/astro-theme-pure)](https://github.com/cworld1/astro-theme-pure/graphs/contributors)

<Aside title='贡献指南'>
贡献 bug 修复或新功能是你能做出的最直接的贡献形式。你最好先提出一个 issue 来讨论你想要做出的改变，这样可以避免不必要的工作，特别是当你想做出破坏性更改时。

所有对 `astro-pure` 包的更改应首先启用 [自定义主题#包模式](/docs/advanced/customize#package-mode)。PR 测试也将在该模式下运行。这要求你首先使用 `bun run yijiansilian` 命令在本地测试你的更改。

至于贡献文档，你可以简单地点击每个文档页面底部的“Edit on GitHub”链接来更改文档内容。
</Aside>
